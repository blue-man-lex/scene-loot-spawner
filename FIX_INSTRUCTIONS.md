## 🧪 **КАК ПРОТЕСТИРОВАТЬ**

### **Шаг 1: Перезагрузка Foundry VTT**
1. Закройте Foundry VTT полностью
2. Запустите Foundry VTT снова
3. Откройте мир с модулем Scene Loot Spawner

### **Шаг 2: Проверка консоли**
1. Откройте консоль разработчика (F12)
2. Найдите сообщения от модуля:
   ```
   SLS | Init
   SLS HAND | Личный коллектор создан
   ```

### **Шаг 3: Тестирование инструментов**
1. **Нажмите на иконку 📦 (Инструменты Лута)** в левой панели
2. **Проверьте появление трех инструментов:**
   - 🎲 **Генератор Лута** (только для ГМ)
   - ✋ **Рука Лута** (для всех)
   - 👁️ **Глаз Лута** (только для ГМ)

### **Шаг 4: Тестирование кликов**
1. **Нажмите на каждый инструмент:**
   - **Генератор Лута**: должно открыться окно генератора
   - **Рука Лута**: должно открыться окно сбора ингредиентов
   - **Глаз Лута**: должно открыться окно просмотра лута

2. **Проверьте консоль:**
   ```
   Loot Spawner: Opening generator window
   Loot Spawner: Opening ingredient hand
   Loot Spawner: Opening loot eye
   ```

### **Шаг 5: Проверка видимости**
1. **Зайдите как игрок (не ГМ):**
   - 🎲 **Генератор Лута** - должен быть скрыт
   - ✋ **Рука Лута** - должен быть виден
   - 👁️ **Глаз Лута** - должен быть скрыт

2. **Зайдите как ГМ:**
   - Все три инструмента должны быть видны

## ⚠️ **ВОЗМОЖНЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ**

### **Проблема: Инструменты не появляются**
**Причина:** Модуль не загрузился правильно
**Решение:**
1. Проверьте консоль на ошибки
2. Убедитесь что модуль включен в настройках мира
3. Перезагрузите Foundry VTT

### **Проблема: Кнопки залипают**
**Причина:** Остались старые обработчики
**Решение:**
1. Полностью перезагрузите Foundry VTT (не F5)
2. Очистите кэш браузера

### **Проблема: Инструменты видны всем**
**Причина:** Неправильная проверка `game.user.isGM`
**Решение:**
1. Проверьте права пользователя в мире
2. Убедитесь что пользователь имеет роль ГМ

### **Проблема: Ошибки в консоли при клике**
**Причина:** Файлы приложений не загрузились
**Решение:**
1. Проверьте наличие файлов:
   - `scripts/lootApp.js`
   - `scripts/hand/ingredientHandApp.js`
   - `scripts/eye/lootEyeApp.js`

## 🔄 **СРАВНЕНИЕ СТАРОГО И НОВОГО ПОДХОДА**

### **Старый подход (проблемный):**
```javascript
// ❌ Конфликтует с V12+
Hooks.on("renderSceneControls", (controls, html) => {
  // jQuery обработчики
  let spawnerButton = $html.find('[data-tool="openSpawnerWindow"]');
  spawnerButton.off('click').on("click", (e) => { ... });
});
```

### **Новый подход (правильный):**
```javascript
// ✅ Совместимо с V12+
Hooks.on("getSceneControlButtons", (controls) => {
  controls.lootSpawner = {
    name: "lootSpawner",
    tools: {
      openSpawnerWindow: {
        name: "openSpawnerWindow",
        button: true,
        onClick: () => new SceneLootApp().render(true)
      }
    }
  };
});
```
