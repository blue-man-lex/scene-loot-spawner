# 🎲 Scene Loot Spawner

[![Foundry VTT Version](https://img.shields.io/badge/Foundry-V13-blue.svg)](https://foundryvtt.com/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![Version](https://img.shields.io/badge/Version-1.0.3-orange.svg)](https://github.com/blue-man-lex/scene-loot-spawner)

> **Мощный инструмент для генерации лута и автоматизации сбора награды на сцене.**

## 🌟 Обзор

Scene Loot Spawner - это универсальный модуль для Foundry VTT, который позволяет мастеру создавать контейнеры с лутом, управлять сбором ингредиентов и организовывать полную автономию в распределении награды между игроками.

### ✨ Ключевые возможности

- 🎲 **Генератор лута** - Создание контейнеров с предметами по профилям и уровням
- 🌿 **Сбор ингредиентов** - Автоматический сбор трав и материалов с карты
- 👁️ **Инструменты мастера** - Удобные инструменты для управления лутом
- 🏪 **Интеграция с THM** - Полная совместимость с Treasure Hoard Manager
- 🎯 **Балансировка по CR** - Автоматическая подстройка под уровень партии
- 🗺️ **Поддержка биомов** - Контекстная генерация для разных локаций

## 🚀 Быстрый старт

### 1. Установка

1. Скачайте модуль из [GitHub Releases](https://github.com/blue-man-lex/scene-loot-spawner/releases)
2. Распакуйте архив в папку `Data/modules/`
3. Активируйте модуль в Foundry VTT
4. Установите зависимости: **Treasure Hoard Manager** и **socketlib**

### 2. Базовое использование

- **Откройте инструменты лута** через панель управления слоем
- **Выберите профиль генерации** (Chests, Barrels, Remains и др.)
- **Настройте уровень сложности** (CR 0-1 до 20)
- **Разместите контейнеры** на сцене
- **Сгенерируйте лут** в выбранных контейнерах

## 📋 Детальное описание возможностей

### 🎲 Генератор лута

#### Профили контейнеров:
- **Chests** - Деревянные сундуки (классика)
- **Crates** - Старые ящики (руины, подвалы)
- **Furniture** - Шкафы и мебель (дома, замки)
- **Remains** - Останки (пещеры, поля битв)
- **Barrels** - Бочки (дocks, склады)
- **Cages** - Клетки (тюрьмы, лаборатории)
- **Treasures** - Сокровищницы (сокровища)
- **Misc** - Разное (универсальные)

#### Уровни сложности (Tier):
- **Tier 0 (CR 0-1)** - Обычные предметы, мало золота
- **Tier 1 (CR 1-4)** - Необычные предметы, базовый лут
- **Tier 2 (CR 5-10)** - Редкие предметы, хороший лут
- **Tier 3 (CR 11-16)** - Очень редкие, ценный лут
- **Tier 4 (CR 17-19)** - Легендарные, эпический лут
- **Tier 5 (CR 20)** - Уникальные, максимальная награда

#### Биомы:
- **Desert** - Пустыня (песок, кости, артефакты)
- **Forest** - Лес (травы, древесина, природное)
- **Mountain** - Горы (руды, камни, драгоценности)
- **Swamp** - Болото (яды, редкие растения)
- **Dungeon** - Подземелья (алхимия, магия)
- **City** - Город (ремесло, торговля)

### 🌿 Сбор ингредиентов

#### Типы ингредиентов:
- **Травы** - Лечебные и ядовитые растения
- **Руды** - Металлы и минералы
- **Ингредиенты** - Алхимические компоненты
- **Редкие материалы** - Особые компоненты для крафта

#### Механика сбора:
- **Визуальные маркеры** - Отображение ингредиентов на сцене
- **Автоматический сбор** - Быстрый сбор в инвентарь игрока
- **Балансировка** - Контроль количества и редкости
- **Интеграция с крафтом** - Совместимость с системами создания

### 👁️ Инструменты мастера

#### Loot Eye (Глаз лута):
- **Визуализация контейнеров** - Подсветка всех лут-объектов
- **Быстрая инвентаризация** - Просмотр содержимого без открытия
- **Статистика лута** - Общая ценность и редкость
- **Массовые операции** - Одновременные действия с группой

#### Ingredient Hand (Рука ингредиентов):
- **Сбор в радиусе** - Автоматический сбор всех ингредиентов рядом
- **Фильтрация по типу** - Выборочный сбор определенных материалов
- **Учет в инвентаре** - Автоматическое добавление игрокам
- **Визуальные эффекты** - Анимации сбора и уведомления

## 🛠️ Конфигурация

### Основные настройки

```javascript
// Источники предметов
game.settings.get('scene-loot-spawner', 'useWorldItems') // Использовать предметы мира
game.settings.get('scene-loot-spawner', 'useCompendiums') // Использовать компендиумы

// Генерация
game.settings.get('scene-loot-spawner', 'defaultMaxItems') // Максимум предметов
game.settings.get('scene-loot-spawner', 'defaultProfile') // Профиль по умолчанию
game.settings.get('scene-loot-spawner', 'defaultTier') // Уровень по умолчанию
```

### Продвинутые настройки

- **Фильтрация по редкости** - Контроль распределения предметов
- **Множители золота** - Настройка количества валюты
- **Весовые коэффициенты** - Балансировка типов предметов
- **Исключения** - Запрет определенных предметов

## 🎮 Использование в игре

### Создание лута на сцене

1. **Активируйте слой** "Инструменты Лута"
2. **Выберите инструмент** "Генератор Лута"
3. **Настройте параметры** в открывшемся окне
4. **Кликните на сцену** для размещения контейнера
5. **Нажмите "Сгенерировать"** для наполнения лутом

### Сбор ингредиентов

1. **Выберите инструмент** "Сбор Ингредиентов"
2. **Настройте радиус** и типы ингредиентов
3. **Кликните на область** для сбора
4. **Просмотрите результаты** в окне сбора

### Управление лутом

1. **Используйте "Глаз Лута"** для просмотра всех контейнеров
2. **Фильтруйте по типу** и редкости
3. **Выполняйте массовые операции** с группой контейнеров
4. **Экспортируйте статистику** для баланса

## 🐛 Устранение неполадок

### Частые проблемы

#### ❌ Контейнеры не создаются

**Решение:**
1. Проверьте установку зависимостей (THM, socketlib)
2. Убедитесь что вы находитесь на слое "Инструменты Лута"
3. Проверьте права доступа (только для ГМ)

#### ❌ Лут не генерируется

**Решение:**
1. Проверьте наличие предметов в компендиумах
2. Убедитесь что профиль и уровень настроены
3. Проверьте фильтры редкости

#### ❌ Ингредиенты не собираются

**Решение:**
1. Проверьте наличие ингредиентов на сцене
2. Убедитесь что игрок выбран
3. Проверьте настройки радиуса сбора
